## 📖 Overview

This project is a **Next.js training application** inspired by a real-world technology company, **LINK Technology Service Providers**.

It focuses on building a modern, scalable frontend using industry-standard tools for data fetching, state management, and styling.

---

## ⚙️ Tech Stack

- **Next.js** – React framework for building fast, scalable applications  
- **SWR** – Data fetching with caching and revalidation  
- **Zustand** – Lightweight state management  
- **Tailwind CSS** – Utility-first styling for rapid UI development  
- **TypeScript** – Type-safe development  

---

## 🚀 Features

- App Router with file-based routing  
- Efficient data fetching using **SWR**  
- Global state management with **Zustand**  
- Fully responsive UI with **Tailwind CSS**  
- Clean and scalable project structure  
- Fast refresh for smooth development experience  

---

## 🧠 Purpose

This project is part of a learning journey to:

- Master modern React and Next.js patterns  
- Work with real-world state management and data fetching tools  
- Build maintainable and scalable frontend architecture  
- Simulate a production-ready company website
